# 买卖条件文档

## 卖出条件 (`judge_sell_id`)

### `sell_id = 1`：标准卖出逻辑

此 ID 包含一套基于多时间周期和技术指标的综合卖出策略，按以下优先级顺序执行：

1.  **清仓信号**：
    - **条件**：周线出现 MACD 顶背离（价格创新高但 DIF 未创新高），并且 日线 K 线图上出现 MACD 死叉（3天内）。
    - **操作**：建议清仓。

2.  **减半信号**：
    - **条件**：日线出现 MACD 顶背离（价格创新高但 DIF 未创新高），并且 日线 K 线图上出现 MACD 死叉（3天内）。
    - **操作**：建议卖出 1/2 仓位。

3.  **阶梯式减仓信号**：
    - **触发条件**：日线 SAR 指标被跌破（3天内）。
    - **操作**：根据周线 RSI 指标的强度执行不同比例的卖出：
        - **周线 RSI > 90**：建议卖出全部剩余仓位。
        - **周线 RSI > 85**：建议卖出剩余仓位的 1/2。
        - **周线 RSI > 80**：建议卖出 1/3 仓位。

## 买入条件 (`judge_buy_id`)

系统支持不同的买入策略，通过 `judge_buy_id` 进行选择。

### `buy_id = 1`：新资金买入信号
- **条件**：日线 RSI < 20 **并且** 周线 RSI < 25。

### `buy_id = 2`：卖出的资金买回的信号（注意只是买回之前卖出部分的资金）
- **条件**：满足以下任一即可：
    1.  日线 RSI < 25 **并且** 周线 RSI < 30。
    2.  日线出现底背离 **并且** 日线 K 线图上出现 MACD 金叉。

### `buy_id = 3`：新资金买入信号
- **条件**：满足以下任一即可：
    1.  日线 RSI < 25 **并且** 周线 RSI < 30。
    2.  日线出现底背离 **并且** 日线 K 线图上出现 MACD 金叉。
    3.  **上证指数 (000001)** 或 **创业板指数 (399006)** 的日线 RSI < 25。

### `buy_id = 4`：新资金买入信号
- **条件**：日线 RSI < 20 **并且** 周线 RSI < 20。
