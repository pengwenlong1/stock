# 买卖条件文档

## 卖出条件 (`judge_sell_id`)

### `sell_id = 1`：标准卖出策略 1

此 ID 包含一套基于多时间周期和技术指标的综合卖出策略，按以下优先级顺序执行：

当SAR跌破死叉时，做以下判断；
1.  **清仓信号 - 周线顶背离**：
    - **条件**：周线出现 MACD 顶背离（参考下方"顶背离判断依据"），并且SAR跌破死叉。
    - **操作**：建议清仓，sell_flag=1

2.  **减仓信号 - 日线顶背离**：
    - **条件**：日线出现 MACD 顶背离（参考下方"顶背离判断依据"），并且SAR跌破死叉。
    - **操作**：建议卖出 1/3 仓位，sell_flag=3

3.  **阶梯式减仓信号**：
    - **触发条件**：SAR跌破死叉。
    - **操作**：根据周线 RSI 指标的强度执行不同比例的卖出：
        - **周线 RSI > 85**：建议卖出剩余仓位的 1/2，sell_flag=2
        - **周线 RSI > 80**：建议卖出剩余仓位的1/3 仓位，sell_flag=3
        - **周线 RSI > 90**：直接清仓，sell_flag=1


### `sell_id = 2`：标准卖出策略 2

此 ID 包含一套基于多时间周期和技术指标的综合卖出策略，按以下优先级顺序执行：

当MACD日线跌破死叉时，做以下判断；
1.  **清仓信号 - 周线顶背离**：
    - **条件**：周线出现 MACD 顶背离（参考下方"顶背离判断依据"），并且SAR跌破死叉。
    - **操作**：建议清仓，sell_flag=1

2.  **减仓信号 - 日线顶背离**：
    - **条件**：日线出现 MACD 顶背离（参考下方"顶背离判断依据"），并且SAR跌破死叉。
    - **操作**：建议卖出 1/3 仓位，sell_flag=3

3.  **阶梯式减仓信号**：
    - **触发条件**：SAR跌破死叉。
    - **操作**：根据周线 RSI 指标的强度执行不同比例的卖出：
        - **周线 RSI > 85**：建议卖出剩余仓位的 1/2，sell_flag=2
        - **周线 RSI > 80**：建议卖出剩余仓位的1/3 仓位，sell_flag=3
        - **周线 RSI > 90**：直接清仓，sell_flag=1

sell_flag是用来判断卖出优先级：1>2>3，也就是如果同时触发，执行优先级高的。

**重要说明**：
- 当多个卖出信号同时触发时，执行最大卖出比例 例如：RSI flag=3（周线RSI>90）触发清仓时，优先级高于RSI>80（只卖出1/3）

## 买入条件 (`judge_buy_id` / `judge_t_id`)

系统支持不同的买入策略，通过 `judge_buy_id`（新开仓）或 `judge_t_id`（做 T 买回）进行选择。两者复用相同的逻辑 ID，但提示信息会有所区别：
- **`judge_buy_id`**：提示 **"新资金买入信号"**。
- **`judge_t_id`**：提示 **"做 T 买回信号"**（用于买回之前卖出的资金）。

### `buy_id` = 1：保守型买入
- **条件**：日线 RSI < 20 **并且** 周线 RSI < 25。

### `buy_id` = 2：标准型买入 / 做 T 买回
- **说明**：此 ID 常用于判断卖出后的资金买回时机。
- **条件**：日线 RSI < 25 **并且** 周线 RSI < 30。

### `buy_id` = 3：指数保护型买入
- **说明**：此 ID 用于监控市场大盘是否处于超卖状态。
- **条件**：**创业板指数 (399006)** 的日线 RSI < 25。

### `buy_id` = 4：极度保守型买入
- **条件**：日线 RSI < 20 **并且** 周线 RSI < 20。

### `buy_id` = 5：指数保护型买入
- **说明**：此 ID 用于监控市场大盘是否处于超卖状态。
- **条件**：**创业板指数 (399006)** 的日线 RSI < 20。

### `buy_id` = 6：上证指数保护型买入
- **说明**：此 ID 用于监控市场大盘是否处于超卖状态。
- **条件**：**上证指数 (000001)** 的日线 RSI < 20。

### `buy_id` = 7：上证指数保护型买入
- **说明**：此 ID 用于监控市场大盘是否处于超卖状态。
- **条件**：**上证指数 (000001)** 的日线 RSI < 25。


说一下买入的规则，judge_buy_id触发的买入，是一次性买入新资金并且买回之前买入之前卖出的资金，
judge_t_id触发的买入，是一次性买入之前卖出的资金而不会买入未进入过市场的新资金
